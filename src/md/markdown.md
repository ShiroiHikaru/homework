# 마크다운 문법

[공식문서링크](https://www.markdownguide.org/getting-started/)

## 마크다운이란?

텍스트 기반의 마크업 언어로 문법이 쉽고 간단하며 HTML과 텍스트 등을 서식 문서로 쉽게 변환이 가능한 문법이다.

### 주 사용처

기본적으로 지원하는 모든 곳에서 사용할 수 있지만 대표적인 예시로 Github 이슈 및 README 편집, 워드프레스, 슬랙, 노션 등의 서비스에서도 사용할 수 있다.

---

## 제목

제목을 만들기 위해서는 문구 앞에 #을 붙인다.
예를 들어 h3과 같은 제목을 만들기 위해서는 #기호 3개를 붙여서 사용한다.

## 예시

# h1 제목

## h2 제목

### h3 제목

#### h4 제목

##### h5 제목

###### h6 제목

h1과 h2는 다음과 같이 줄 형태로도 작성할 수 있다.

## 예시

# H1 > 대제목 1 예시

## H2 > 부제목 2 예시

#으로 사용 시 #쓰고 한칸 띄워서 사용한다.

####안녕하세요 (X)

#### 안녕하세요 (O)

## 인용문

'>' 문자를 사용한다.

#### 인용문 1개만 작성 시

> 이것은 인용문 입니다.

#### 여러개의 인용문을 작성 시

> 이것은 인용문 입니다.
>
> > 인용문에 인용문을 추가했습니다.
> >
> > > 인용문을 3개 추가 했습니다.

인용문 안에는 다른 마크다운 요소를 포함할 수 있다

## 예시

> ### 인용문 제목
>
> 인용문을 쓸 때는 별도의 '>'기호를 넣어줍니다.

## 예시(중첩 블록 인용문)

> ### 인용문 제목
>
> > 개발 공부는 어렵지만 꼭 해내야 해!

## 글자 변경

#### Bold 체

강조하고 싶은 문구 양 옆에 '\*\*'를 붙여준다.

예시) 저는 **떡볶이를 제일** 좋아합니다.

예시) 이 **링크**를 클릭해주세요.

---

#### 이탤릭 체

이탤릭 체를 적용하고 싶은 문구 양 옆에 '\*' 를 붙여 준다.

예시) _고양이는 야옹 소리를 낸다._

예시) _포메라니안은 눈코입이 까만콩과 닮았다._

---

#### 볼드와 이탤릭을 동시에 적용하고 싶을 경우

강조하고 싶은 문구 양 옆에 '\*\*\*'를 추가한다.

예시) 해당 제품은 **_물에 매우 취약_** 합니다.

예시) 어린이들이 **_먹지 않도록_** 주의하십시오.

#### 취소선

'~'표시로 취소선을 긋고 싶은 문구 양옆에 추가해 준다.

예시) ~~치킨 먹고싶다~~

---

#### 글상자

텍스트 전체를 '`' 기호로 감싼다.

예시)
`내일 친구들과 김포의 유명한 카페에 방문하기로 했어요.`

## 순서(List)

'ol'과 'ul'을 사용하지만 순서가 있는 항목은 숫자와 마침표, 순서가 없는 항목은 '\*', '+', '-' 기호로도 지원할 수 있다.

### 순서가 있는 항목

1. 사과
2. 바나나
3. 키위
4. 딸기

### 순서가 없는 항목

- 프론트엔드 개발 공부 항목
  - HTML
    - 시맨틱태그
    - div태그
    - input태그
  - CSS
    - property{value}
    - flex
    - grid
  - Javascript
    - 변수
    - 반복문
    - if문
- 백앤드 입문

## 링크

링크는 [이름](링크) 구조로 작성하여 추가 할 수 있다.

1. [네이버](www.naver.com)
2. [유튜브](https://www.youtube.com/@coohde/videos)

참조링크를 그대로 사용할 수도 있다.
문서 내 일반 url이나 꺾쇠 괄호(<>) 안의 url은 자동으로 링크를 사용한다.

예시) [엽떡 홈페이지]: https://www.yupdduk.com/
